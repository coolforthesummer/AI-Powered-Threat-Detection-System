# Phase 2: Data Collection and Preprocessing

## Goal

The goal of Phase 2 is to prepare a clean, labeled, and model-ready dataset for Phase 3.  
By the end of this phase, the team should have:

- selected the main dataset source,
- documented the meaning of each feature,
- cleaned and standardized the raw data,
- created a preprocessing pipeline,
- exported training-ready files for model development.
