# AI-Powered-Threat-Detection-System
AI-Powered Threat Detection System

## Project Overview

The goal of this project is to build an AI-powered system that can detect threats like DDoS, phishing, or malware attacks in real-time by analyzing network traffic or user behavior data. The system will utilize machine learning algorithms for anomaly detection and will be integrated with a backend API built with Flask.
